Detection Coverage

Every Major AML Typology Covered

FRAUD.GUARD is aligned with FATF guidance and FinCEN reporting requirements. From placement to integration — and everything in between.

🔄

Layering (Obfuscation)

Cycle detection via CSR adjacency + bounded DFS. Identifies A→B→C→A circular flows. Bridge-node centrality reveals connectors.

Engine: Graph · Cycle DFS
📥

Placement (Structuring)

Fan-in detection + structuring sequence analysis. Flags repeated just-below-CTR-threshold transactions ($9k–$10k) that avoid FinCEN reporting.

Engine: Graph + Temporal
📤

Integration (Dispersion)

Fan-out detection combined with temporal concentration analysis. Single source → many targets in a narrow time window = integration signature.

Engine: Graph + Temporal
🤖

Money Mule Networks

GNN node classifier identifies mule-cluster patterns — multi-hop accounts with dormancy-then-spike behavioral profiles and unusual counterparty diversity.

Engine: GNN + Behavioral
🏦

Smurfing

Temporal concentration + volume acceleration detectors flag coordinated low-amount transactions across multiple accounts in synchronised bursts.

Engine: Temporal + LSTM
🆔

Synthetic Identity Fraud

Thin-file accounts with sudden utilisation spikes. Behavioral shift detector + LSTM emerging-link predictor flag relationship formation before the typology fully emerges.

Engine: Behavioral + Temporal
🔐 Regulatory alignment: FinCEN CTR ($10k threshold) · FATF Risk-Based Approach · SAR filing workflow

How the Adversarial Loop Works

1
PPO Agent Attacks
A Proximal Policy Optimisation agent generates realistic money-laundering patterns (peel chains, smurfing, mule networks) specifically designed to evade detection.

Detection Engines Score
The pipeline scores adversarial transactions. The agent's reward is — (detection_score): it is rewarded for evading detection, penalised for being caught.

Models Retrain on Failures
When the adversarial agent succeeds, we add its patterns to the training corpus. The detectors retrain. Evasion gets harder with every cycle.
