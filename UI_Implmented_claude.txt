 What was implemented

  New entry point

  app.py — Unified multi-page Streamlit app (streamlit run app.py). Loads shared CSS (font, glassmorphism cards, risk-tier badges, accessible text symbols) once
  for all pages. Uses st.navigation so navigation appears in the sidebar automatically.

  ---
  pages/graph_explorer.py (new)

  - Interactive Plotly scatter graph replacing the static PNG from visualizer.py
  - 5 risk-tier colours with text symbols in legend (not binary red/blue)
  - Node size = connection degree
  - Hover tooltip: account, tier, score, in/out degree
  - Searched account gets a white highlight border
  - Risk-tier multiselect filter; max-node slider (highest-risk first)
  - Account detail panel shown below graph when a node is searched
  - Labels auto-suppressed when >80 nodes

  ---
  pages/account_detail.py (new)

  - Account selector with tier tag + alert-state icon in label
  - Overview table when no account is selected
  - Risk score gauge with animated fill bar
  - Active signal list with per-phase icon, colour, and phase label
  - Alert management: Unreviewed / Investigating / False Positive / Escalated + free-text notes, persisted to alert_state.db
  - Transaction timeline: incoming vs outgoing bar chart, date-range filterable
  - Counterparty mini-graph: interactive Plotly, nodes coloured by their risk tier

  ---
  src/alert_store.py (new)

  SQLite-backed store for alert states. get_alert_state(), set_alert_state(), get_all_states().

  ---
  src/visualizer.py (updated)

  - Added build_plotly_graph() — full interactive Plotly figure, same API as the old plot_graph but returns a go.Figure instead of saving a PNG
  - Added plot_graph_with_tiers() — improved static PNG with 5-tier colours, node labels re-enabled, edge amount labels, and a proper matplotlib legend
